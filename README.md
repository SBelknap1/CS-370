# CS-370
This my CS-370 Repo 

For this project, I trained a reinforcement learning agent to navigate a pirate treasure map. The starter code provided the environment, rewards, and overall structure. I wrote the Q‑learning logic, updated the Q‑table, tuned the learning parameters, and completed the training loop so the agent could learn an optimal path.

What I Learned
This project showed how reinforcement learning fits into the larger field of computer science. Computer scientists build systems that solve problems, automate tasks, and learn from data. My approach was to break the problem into steps, test often, and adjust the algorithm based on the agent’s behavior.

Ethical Responsibilities

As a computer scientist, I’m responsible for writing code that is reliable, fair, and safe for users. That includes protecting data, avoiding bias, and making sure the system behaves as expected for both the user and the organization.
